Welcome!

So you've stumbled onto a work in progress.  This is the home of Nerduino IOT, an open source environment for developing and managing networked-Arduino devices.

Nerduino IOT Goals

- Provide standard gateways for Arduino IOT devices to interact with one another

- Provide an exceptionally simple IOT API

- Minimize latency between IOT updates

- Provide a minimal memory footprint behind the API

- Provide a minimal processor and RAM overhead

- Manage a collection of IOT devices

- Expose services to IOT devices

- Provide a web interface to IOT devices.
